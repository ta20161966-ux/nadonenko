## Описание программы

*Expense Tracker* — это простое кроссплатформенное приложение с графическим интерфейсом (GUI) для учёта личных финансов. Программа позволяет записывать ежедневные расходы, категоризировать их и анализировать траты за выбранный период.

**Основные возможности:**
*   **Добавление расходов:** ввод суммы, категории и даты.
*   **Сохранение данных:** вся информация надёжно хранится в файле `data/expenses.json`.
*   **Фильтрация:** быстрый поиск расходов по категории и/или диапазону дат.
*   **Подсчёт суммы:** автоматический расчёт общих трат на основе применённых фильтров.

Приложение написано на языке **Python** с использованием библиотеки **Tkinter**.

## Как использовать

1.  **Установка:**
    1.  Убедитесь, что на вашем компьютере установлен **Python** версии 3.8 или новее.
    2.  Скачайте или клонируйте этот репозиторий на свой компьютер.

2.  **Запуск:**
    1.  Откройте терминал (командную строку) в папке с проектом.
    2.  Запустите приложение командой:
        ```bash
        python expense_tracker.py
        ```

3.  **Работа с интерфейсом:**
    *   Введите сумму расхода, выберите категорию (например, *«Еда»*, *«Транспорт»*) и укажите дату в формате `ГГГГ-ММ-ДД` (например, `2026-04-25`).
    *   Нажмите кнопку **«Добавить расход»**.
    *   Запись появится в таблице сверху.

## Примеры использования

### 1. Добавление нового расхода

*   В поле **Сумма** введите: `1200`
*   В поле **Категория** введите: `Аренда`
*   В поле **Дата** введите: `2026-04-01`
*   Нажмите кнопку **«Добавить расход»**.

### 2. Фильтрация расходов по категории

Чтобы посмотреть только траты на еду:
1.  Введите в поле **Фильтр по категории**: `еда`.
2.  Нажмите кнопку **«Применить фильтр»**.
3.  В таблице отобразятся только те строки, где в колонке «Категория» есть слово «еда».

### 3. Анализ трат за определённый период

Чтобы узнать, сколько денег было потрачено в апреле:
1.  В поле **Дата с**: введите `2026-04-01`.
2.  В поле **Дата по**: введите `2026-04-30`.
3.  Нажмите кнопку **«Применить фильтр»**.
4.  В строке «Итого» внизу окна отобразится общая сумма всех расходов за указанный месяц.
